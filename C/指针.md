指针变量实际存放的是某个变量的地址，使用指针时指针会指向那个地址，从而取得需要的值

指针变量的形式

int* p;

int *p;

int *p,q;

int *p=&i;

不能用*p=&i;的形式来给指针进行赋值

利用指针可以让函数内部能够访问和修改函数外部的值

传地址给函数时要使用函数名(&i)的形式

*p可以简单理解为一个变量来用，当做a的形式，其他规则也相差无几，能够对他做运算或者打印出来
