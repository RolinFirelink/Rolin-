- CSS介绍

CSS是Cascading Style Sheets的简写，其中文全称为层叠样式表。是一种用于设置和布局网页的一种计算机语言，可以告知浏览器如何渲染解析界面元素。作为后端的程序员，我们不需要对CSS有太多的深入了解，稍微知道那点就差不多得了，所以我们这里的学习也是从简
