具体我们可以构造代码如下

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>RegExp</title>

<body>

</body>
<script>
    //1.验证手机号
    //规则：第一位1，第二位358，第三到十一位必须是数字。总长度11

    //2.验证用户名
    //规则：字母、数字、下划线组成。总长度4~16

</script>
</html>
```

```
    // set(key,value)  添加元素

    }
```
